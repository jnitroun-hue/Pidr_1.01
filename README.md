# P.I.D.R.

Карточная игра с ботами, онлайн-комнатами, рейтингом, NFT-коллекцией и маркетплейсом.

Доступна как **Telegram Mini App**, **VK Mini App** и в **браузере**.

## О игре

P.I.D.R. — многопользовательская карточная игра с классическими правилами, несколькими стадиями партии и режимом против ботов. Можно собирать коллекцию NFT-карт, торговать на маркетплейсе, участвовать в рейтинге и еженедельных призах.
